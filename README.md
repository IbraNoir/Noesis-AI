# Noesis AI

Noesis AI — это интеллектуальный Telegram-ассистент, созданный для общения, помощи в обучении, анализа текста и генерации кода.  
Проект помогает пользователю экономить время, быстрее работать с информацией и получать понятные ответы на вопросы.

## Возможности

- Общение на каждый день
- Ответы на вопросы
- Анализ текста
- Генерация кода
- Помощь с заданиями
- Создание и улучшение текстов
- Анализ данных
- Поддержка русского языка
- Поддержка английского языка при необходимости

## Цели проекта

- Упростить работу с текстом и информацией
- Помогать в обучении
- Автоматизировать часть поддержки и ответов
- Помогать с программированием
- Создать удобного AI-помощника для Telegram

## Будущие функции

- Память диалога
- Режимы работы: чат, код, текст, обучение
- Анализ файлов
- Поддержка голосовых сообщений
- Поиск информации в интернете
- Веб-версия

## Технологии

Пока проект находится в разработке.  
Планируемые технологии:

- Python
- Telegram Bot API
- aiogram или python-telegram-bot
- SQLite
- AI API / LLM
- FastAPI (возможная будущая веб-часть)
